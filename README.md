<b>Introduction</b><br>
"Horizontal Scroll UI: A concept landing page designed to showcase high-resolution imagery and typography. This project leverages CSS Overflow properties to create distinct horizontal scrolling zones for content and images, featuring custom Webkit scrollbar styling and a flexible Flexbox layout."

<b>Features</b><br>
*orizontal Scrolling Layouts: Implements overflow-x: auto to create swipeable, horizontally scrolling sections for both text headlines and image galleries.

*Custom Scrollbar Styling: Features advanced CSS customization of the ::-webkit-scrollbar pseudo-elements to create a sleek, branded scrollbar experience.

*Flexbox Architecture: Utilizes CSS Flexbox for precise alignment of navigation elements (nav1, nav2) and content distribution.

*Visual Hierarchy: Establishes a clear layout structure with distinct zones for top-level navigation, category menus, and feature content.

*Fluid Image Handling: Uses object-fit: cover to ensure images maintain their aspect ratio while filling the horizontal scrolling container.
